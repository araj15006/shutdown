# Shutdown/Restart GUI

This simple GUI application allows users to easily perform shutdown, restart, and sleep actions on their computer. It is built using Tkinter in Python.

## Features

- **Shutdown:** Click the "Shutdown" button to initiate a system shutdown.
- **Restart:** Click the "Restart" button to restart the computer.
- **Sleep:** Click the "Sleep" button to put the computer to sleep.

## How to Use

1. Ensure you have Python installed on your system.
2. Clone this repository to your local machine.
3. Run the script using `python shutdown_restart_gui.py`.
4. Use the corresponding buttons to perform the desired action.
